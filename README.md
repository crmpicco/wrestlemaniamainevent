WrestleMania Main Event
=====================

A fan site built on the JavaScript framework AngularJS (for HTML data-binding) and Twitter Bootstrap (for RWD) to showcase the main events of the pro wrestling promotion WWE's main pay-per-view - WrestleMania - over the past 30 years.
